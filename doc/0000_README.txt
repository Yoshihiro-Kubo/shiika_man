詩歌マネージャ
￣￣￣￣￣￣￣

1. はじめに
　詩歌マネージャは、俳句・川柳・都都逸・和歌、等の「詩歌」をマネージメントするソフトです。
　コンセプトについては、doc/0010_コンセプト.txt をご参照ください。


2. ディレクトリ構造
	code	ソースコード・ビルド用スクリプトを格納
	doc		ドキュメントを格納

2.1 code ディレクトリの内容
　ソースコード
	shiika_man.c	詩歌マネージャソースコード
	shiika_man.h	詩歌マネージャヘッダーファイル
	moji.h			文字関連ヘッダーファイル

　ビルド用スクリプト
	comp			ビルド用スクリプト
	compile			ビルド用スクリプト（内部）

2.2 doc ディレクトリの内容
	0000_README.txt				本ファイル
	0010_コンセプト.txt			詩歌マネージャのコンセプト
	0020_コーディング資料.txt	コーディングのための検討資料（参考）
								※ 古い情報・変更してしまった情報が残っているかもしれません。
								　 そのつもりで読んでください。

3. ビルドのための情報
3.1 開発環境
	環境						Windows11 WSL2 Ubuntu 22.04.2 LTS
	コンパイラ					gcc 11.3.0
	多倍長整数演算ライブラリ	GNU MP 6.1.2

3.2 ビルド方法
　ビルド用スクリプト comp および compile を参照にしてください。


4. ライセンス
　詩歌マネージャのライセンスは GPL v3 or later です。


5. 注意事項
　Ubuntu 以外でも、gcc と GNU MP が使える環境であればコンパイルおよび実行ができるはずではあります。
　しかし、試していません。

　ただし、10 の 24 乗を示す単位の「𥝱」（「禾+予」）はシフト JIS に含まれていないため、コードページ 932 の環境では表示が乱れる可能性があります。
　その場合、コードページを 65001(UTF-8) に設定すると改善されるかもしれません。（未確認）

　less -x4 とかで見てください。
